# slowroute

A UI/UX focused redesign of [slowroute](https://mrt-map.github.io/slowroute), a navigator for the [Minecart Rapid Transit server](https://minecartrapidtransit.net) powered by TypeScript. Data comes from [gatelogue](https://github.com/mrt-map/gatelogue). Thanks to Kev Quirk for [simple.css](simplecss.org).
